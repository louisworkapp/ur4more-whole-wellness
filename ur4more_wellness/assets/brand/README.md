# Brand Assets

This directory contains the brand assets for UR4MORE Wellness app.

## Required Files:

### App Icons
- `app_icon.png` - Main app icon (1024x1024px)
- `mark.png` - Brand mark for app bars (48x48px)
- `appbar_logo.png` - Logo for app bars (96x32px)

### Splash Screen
- `splash_mark.png` - Main splash screen icon (256x256px)
- `wordmark.png` - Brand wordmark for splash (200x60px)

## Design Guidelines:

### Color Palette
- Primary: #1F4ED3 (Indigo)
- Accent: #7FA4FF (Light Blue)
- Mint: #3AD1C3 (Teal)
- Background: #0E1420 (Dark Ink)
- Surface: #1A2234 (Card Background)

### Style
- Ancient-modern aesthetic
- Clean, geometric shapes
- Subtle gradients and shadows
- Premium dark theme
